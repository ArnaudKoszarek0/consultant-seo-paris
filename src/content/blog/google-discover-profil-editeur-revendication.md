Google s'apprête à lâcher une fonctionnalité que les éditeurs attendent depuis des années. Des indices trouvés dans le code de Google Discover montrent qu'il sera bientôt possible de **revendiquer son profil éditeur** directement dans le flux Discover. Et ça change pas mal de choses pour le SEO.

## Ce qu'on a trouvé dans le code

Le développeur Damien (andell) a repéré une ligne de code dans le flux Discover : `discover_is_profile_claimed_`. Cette variable est suivie soit d'un identifiant MID (l'entité Google associée à votre marque), soit du domaine si l'entité n'existe pas.

En clair : Google prépare un système où les éditeurs pourront **revendiquer leur profil Discover**, un peu comme on revendique un Knowledge Panel ou un profil Google Business. Le paramètre apparaît aussi dans les cartes du flux, ce qui laisse penser que les utilisateurs verront directement si un profil a été revendiqué ou non.

## Pourquoi c'est un signal fort

Google Discover génère entre **30% et 50% du trafic** de certains sites médias. Pourtant, jusqu'ici, les éditeurs n'avaient quasi aucun contrôle sur la façon dont ils apparaissaient dans le flux. Pas de personnalisation du profil. Pas de modification des informations affichées. Rien.

Google News avait déjà son Publisher Center. Google Search a les Knowledge Panels revendicables. Google Maps a les fiches GBP. Discover restait le grand oublié. Ce n'est plus le cas.

Un profil Discover revendiqué pourrait permettre de :

- Modifier les informations affichées sur votre page éditeur dans Discover
- Renforcer la confiance de Google envers votre entité (signal E-E-A-T direct)
- Contrôler votre image de marque dans le flux, plutôt que de laisser Google décider seul
- Se différencier visuellement des éditeurs qui n'ont pas revendiqué leur profil

## Le lien avec l'entité Google (et pourquoi c'est le moment de s'en occuper)

Le code mentionne le MID entity. C'est l'identifiant unique de votre marque dans le Knowledge Graph de Google. Si Google lie le profil Discover à votre entité, alors les sites qui ont un Knowledge Panel bien structuré partiront avec un avantage net.

Mon conseil : vérifiez dès maintenant si votre marque a une entité dans le Knowledge Graph. Tapez votre nom de marque dans Google. Si un Knowledge Panel apparaît à droite, vous avez une entité. Sinon, il va falloir travailler dessus. Structurez vos données. Créez ou nettoyez votre page Wikidata. Alimentez votre présence sur les sources que Google utilise pour construire le Knowledge Graph.

Les sites qui n'ont pas d'entité MID se retrouveront identifiés par leur domaine seul. C'est moins fort. Moins fiable aux yeux de Google. Et probablement moins mis en avant dans Discover.

## Ce que ça implique pour votre stratégie Discover

Premier point : ne dormez pas sur cette fonctionnalité. Quand Google ouvrira la revendication des profils Discover, les premiers à revendiquer auront un avantage. C'est ce qu'on a vu avec les Knowledge Panels. C'est ce qu'on a vu avec les fiches GBP. Les early adopters captent plus de visibilité.

Deuxième point : le badge "profil revendiqué" visible dans le flux pourrait devenir un facteur de clic. Les utilisateurs font davantage confiance aux sources vérifiées. On l'observe déjà sur YouTube avec les coches de vérification. Un profil revendiqué dans Discover pourrait jouer le même rôle.

Troisième point : ça confirme la direction que prend Google. Les signaux d'entité et de marque pèsent de plus en plus lourd. Google veut savoir qui publie, pas seulement quoi. Revendiquer son profil Discover s'inscrit dans cette logique d'authentification des sources.

## Ce qu'il faut faire maintenant

Concrètement, la fonctionnalité n'est pas encore en production. Mais voici ce que vous pouvez préparer dès aujourd'hui :

1. **Vérifiez votre entité Knowledge Graph** via une recherche de marque sur Google
2. **Nettoyez votre page Wikidata** si elle existe, créez-la si elle n'existe pas
3. **Structurez vos données Schema.org** avec Organization et les propriétés sameAs
4. **Surveillez le Publisher Center** de Google News, souvent le point d'entrée pour ce type de revendication
5. **Optimisez vos contenus Discover** : images de qualité (min. 1200px de large), titres engageants sans clickbait, contenu frais et original

Google Discover est en train de passer d'un canal passif à un canal que les éditeurs peuvent activement optimiser. La revendication de profil est une brique de plus dans cette évolution. Et si vous attendez que tout le monde s'y mette pour bouger, il sera trop tard.
